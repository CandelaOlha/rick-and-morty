# Rick and Morty

## About this project

This is the first project I've created with React. It is a very simple page in which you can find information about characters and episodes of the famous series "Rick and Morty".

## How to start

To access the project you must run the following command in your console:

`npm start`

## Technologies

- React
- Sass
